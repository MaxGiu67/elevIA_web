# UC-01. RAG Knowledge Base Aziendale

## Sintesi

Sistema di interrogazione in linguaggio naturale. I nostri clienti fanno domande e ricevono risposte dai documenti aziendali, con riferimento alle fonti.

## Parametri

| Parametro | Valore |
|-----------|--------|
| Area | Knowledge |
| Effort | 15 gg/persona |
| Accuracy target | ≥85% |
| Latenza | < 5 secondi |

## Problema

Le organizzazioni accumulano conoscenza in posti diversi: file server, SharePoint, wiki, email. I dipendenti perdono tempo a cercare, spesso non sanno che l'informazione esiste. La conoscenza esce dall'azienda quando le persone se ne vanno.

## Soluzione

Un'interfaccia conversazionale che interroga tutto il patrimonio documentale. L'utente fa domande, il sistema recupera i documenti rilevanti, genera una risposta sintetica e cita le fonti.

## Benefici

- Riduzione tempo ricerca > 70%
- Risposte consistenti basate su documentazione ufficiale
- Conoscenza preservata
- Onboarding accelerato
