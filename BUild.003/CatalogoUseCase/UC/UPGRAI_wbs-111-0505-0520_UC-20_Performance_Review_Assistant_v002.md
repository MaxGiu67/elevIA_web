# UC-20. Performance Review Assistant

## Sintesi

Sistema di supporto per le performance review. Raccoglie feedback, aggrega dati, suggerisce punti di discussione, genera bozze di valutazione. Il manager arriva al colloquio preparato e con dati oggettivi.

## Parametri

| Parametro | Valore |
|-----------|--------|
| Area | HR |
| Effort | 15 gg/persona |
| Accuracy target | ≥85% |
| Latenza | < 10 secondi |

## Problema

Le performance review sono un incubo: manager impreparati, feedback generico, valutazioni soggettive, documenti fatti all'ultimo minuto. I dipendenti le percepiscono come inutili o ingiuste. HR raccoglie documenti incompleti. Il valore della review si perde.

## Soluzione

Un assistente che prepara il manager alla review: raccoglie feedback dai colleghi, aggrega dati di performance, suggerisce punti di discussione, genera una bozza strutturata. Il manager rivede, personalizza e conduce un colloquio di valore.

## Benefici

- Tempo preparazione -70%
- Review basate su dati
- Feedback consistente in tutta l'organizzazione
- Dipendenti che percepiscono equità
- HR con documentazione completa
