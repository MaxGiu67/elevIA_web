# Prb.05 — Report Manuali

## Il problema

Il nostro cliente produce report settimanali, mensili, trimestrali. Raccolta dati da N fonti, aggregazione, formattazione, revisione. Ore di lavoro ripetitivo per persone qualificate.

## Come si manifesta

Checklist per l'Assessment:

- [ ] Analisti che passano il lunedì sui report settimanali
- [ ] Excel con formule che si rompono
- [ ] Report consegnati in ritardo
- [ ] Errori scoperti dopo la distribuzione
- [ ] Formati inconsistenti
- [ ] Impossibile fare report ad-hoc veloci

## Impatto

| Area | Conseguenza |
|------|-------------|
| Tempo | Ore/uomo su data gathering |
| Decisioni | Dati in ritardo rispetto al momento giusto |
| Qualità | Errori umani non prevenuti |
| Costo opportunità | Analisti che potrebbero fare analisi vere |

## Use Case applicabili

**UC-09 Report Automatici** (primario)
Raccoglie dati, aggrega, produce report in formato standard. Automatico.

**UC-03 Sintesi Riunioni** (secondario)
Genera verbali automatici che alimentano i report di progetto.

## Risultato atteso

Tempo produzione report -90%. Zero errori trascrizione. Dati real-time invece che periodici.
