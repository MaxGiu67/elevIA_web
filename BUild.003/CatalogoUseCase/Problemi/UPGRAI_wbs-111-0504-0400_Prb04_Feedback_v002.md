# Prb.04 — Feedback Non Analizzati

## Il problema

Il nostro cliente raccoglie feedback ovunque: recensioni, sondaggi, social, email, chiamate. Li archivia e basta. Insight preziosi restano sepolti in dati mai elaborati.

## Come si manifesta

Checklist per l'Assessment:

- [ ] NPS raccolto ma analizzato superficialmente
- [ ] Recensioni negative scoperte per caso
- [ ] Trend di insoddisfazione identificati troppo tardi
- [ ] Marketing che non conosce il sentiment reale
- [ ] Prodotto scollegato dalla voce del cliente
- [ ] Crisi reputazionali senza preavviso

## Impatto

| Area | Conseguenza |
|------|-------------|
| Opportunità | Testimonial positivi non identificati |
| Rischi | Criticità che diventano crisi |
| Strategia | Decisioni scollegate dal cliente |
| Competitività | Nessun confronto con competitor |

## Use Case applicabili

**UC-08 Analisi Sentiment** (primario)
Analizza tono e tematiche automaticamente. Identifica trend e alert.

## Risultato atteso

Visibilità real-time sul sentiment. Alert automatici per negativi. Tempo ridotto tra feedback e azione.
