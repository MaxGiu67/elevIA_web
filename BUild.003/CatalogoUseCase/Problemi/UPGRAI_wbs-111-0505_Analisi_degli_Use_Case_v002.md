# Analisi degli Use Case

## Scopo

Questa sezione documenta i 20 Use Case UPGRAI. Per ogni Use Case trovi tre documenti con scopi diversi.

## Struttura documentale

| Documento | Destinatario | Contenuto |
|-----------|--------------|-----------|
| **Scheda Cliente** | Cliente finale | Problema, soluzione, benefici concreti. Linguaggio business, zero tecnicismi. Possiamo personalizzarla con il tuo brand. |
| **Scheda Tecnica** | Cliente + nostro team Sales | Scope, esclusioni, prerequisiti, timeline, criteri di accettazione. Serve per quotare e pianificare. |
| **Specifiche Tecniche** | Tuo CTO / Tech Lead | Architettura, competenze richieste, certificazioni, piano di delivery, rischi, checklist go-live. |
