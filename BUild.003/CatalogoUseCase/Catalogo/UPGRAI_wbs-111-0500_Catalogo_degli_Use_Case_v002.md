# Catalogo Use Case UPGRAI

## A cosa serve

Il Catalogo contiene 20 applicazioni AI standardizzate che proponiamo ai nostri clienti. Ogni Use Case è documentato con tre livelli di dettaglio per scopi diversi.

## Come è organizzato

**Livello 1 — Navigazione**
- Questo documento (mappa del catalogo)
- Elenco Use Case
- Griglia Problemi/Soluzioni

**Livello 2 — Problemi**
Dieci schede che descrivono i problemi aziendali più comuni. Vi aiutiamo a identificare cosa proporre durante l'Assessment.

**Livello 3 — Use Case**
Per ogni Use Case, tre documenti:
- Scheda Cliente → per vendere al cliente finale
- Scheda Tecnica → per definire scope e fattibilità
- Specifiche Tecniche → per guidare il nostro team tecnico

| Area | Use Case |

## Parametri standard

Tutti gli Use Case hanno parametri uniformi per semplificare pianificazione e delivery.

| Parametro | Valore |
|-----------|--------|
| Effort | 15 gg/persona |
| Accuracy | ≥85% |
| Latenza | < 5 secondi |
| Disponibilità | ≥99% |
